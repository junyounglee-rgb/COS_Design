=====================================================
  UI String 브랜치 전파 도구 - 왕초보 가이드
=====================================================

[ 이 도구가 뭔가요? ]

  ui_strings.xlsx 에 새 스트링을 추가한 뒤,
  여러 브랜치에 일일이 들어가서 복붙하는 작업을
  버튼 하나로 자동으로 해주는 도구입니다.

  자동으로 해주는 것:
    - datasheet.exe 실행 (pb 파일 생성)
    - 선택한 모든 브랜치에 xlsx + pb 커밋
    - GitHub push

-----------------------------------------------------

[ 최초 설치 (한 번만) ]

1. install.bat 더블클릭
   → 필요한 프로그램 자동 설치
   → "설치 완료!" 메시지가 나오면 성공

   (Python이 없다면 먼저 설치하세요)
   https://www.python.org/downloads/
   설치 시 "Add Python to PATH" 반드시 체크!

2. run_propagate.bat 더블클릭 → 브라우저에서 앱 열기

3. [설정] 섹션에서 아래 항목 입력 후 [💾 설정 저장] 클릭
   (한 번 저장하면 이후에는 자동으로 불러옵니다)

   - 저장소 경로 (cos-data 루트)
     예) D:\COS_Project\cos-data

   - datasheet.exe 경로
     예) D:\COS_Project\cos-data\datasheet.exe

   - 연동 저장소 경로 (한 줄에 하나씩)
     예)
       D:\COS_Project\cos-common
       D:\COS_Project\cos-client

   - 작업자 이름
     예) 홍길동

-----------------------------------------------------

[ ★ 필수 작업 순서 ★ ]

  ┌─────────────────────────────────────────────────┐
  │  1️⃣  풀 받기  →  2️⃣  엑셀 작업  →  3️⃣  전파 실행  │
  │                                                 │
  │  반드시 이 순서를 지키세요!                        │
  │  엑셀 작업을 먼저 하면 충돌이 발생할 수 있습니다.    │
  └─────────────────────────────────────────────────┘

-----------------------------------------------------

[ 매번 사용할 때 - 순서 ]

  STEP 1. run_propagate.bat 더블클릭
          → 브라우저에서 프로그램이 열립니다

  STEP 2. 전파할 브랜치 선택
          ※ main 은 필수 선택 (해제 불가)
          ※ main 이 항상 가장 먼저 처리됩니다

  STEP 3. [🔽 선택 브랜치 풀 받기] 클릭  ← 먼저!
          → 선택한 모든 브랜치의 최신 코드를 받아옵니다
          → 완료될 때까지 기다리세요

  STEP 4. ui_strings.xlsx 열어서 스트링 추가/수정
          (풀 받기 완료 후에 엑셀 작업을 시작하세요)

  STEP 5. 엑셀 저장 후 닫기
          ※ 전파 실행 전에 반드시 Excel 파일을 닫으세요

  STEP 6. "ui_strings.xlsx 변경 사항" 확인
          추가/변경/삭제된 키 목록이 표시됩니다
          목록이 안 보이면 [🔄 새로고침] 클릭

  STEP 7. 작업자 이름 및 커밋 메시지 확인

  STEP 8. [🚀 string 파일 전파] 클릭
          → xlsx 복사 + datasheet 실행 + 커밋 + 푸시 자동 진행

  STEP 9. 결과 확인
          ✅ 성공 / ❌ 실패 (실패 시 원인 표시)

  종료: 열린 검은 창(터미널)을 닫으세요

-----------------------------------------------------

[ 주의사항 ]

  ⚠ 실행 전에 ui_strings.xlsx 를 닫아주세요.
    Excel이 파일을 열고 있으면 읽기 오류가 납니다.

  ⚠ main 브랜치는 항상 포함됩니다.
    main 에 올리기 싫은 내용이면 이 도구를 쓰지 마세요.

  ⚠ 도중에 창을 닫지 마세요.
    전파가 완료되기 전에 닫으면 일부 브랜치만
    커밋된 상태가 될 수 있습니다.

  ⚠ 두 명이 동시에 실행하지 마세요.
    같은 브랜치에 동시에 push 하면 충돌납니다.

-----------------------------------------------------

[ 뭔가 잘못됐을 때 ]

  ❌ "변경된 내용이 없습니다" 라고 뜰 때
     → xlsx 를 저장했는지 확인하세요
     → [🔄 새로고침] 버튼을 눌러보세요

  ❌ push 실패가 떴을 때
     → 오류 상세를 펼쳐서 내용을 확인하세요
     → 네트워크 문제라면 잠시 후 다시 시도하세요
     → 브랜치 보호 규칙 위반이면 담당자에게 문의

  ❌ datasheet.exe 실패가 떴을 때
     → xlsx 에 잘못된 값이 입력됐을 가능성이 높습니다
     → 오류 메시지에서 어떤 파일/컬럼인지 확인하세요

  ❌ 잠금 파일 관련 오류가 떴을 때
     → 이전 실행이 비정상 종료된 것입니다
     → 설정 섹션 하단의 [🗑️ 잠금 파일 삭제] 버튼을 누르세요

-----------------------------------------------------

[ 브랜치 목록 변경하는 법 ]

  GUI 에서 [⚙️ 브랜치 목록 편집] 버튼 클릭
  → YAML 형식으로 직접 편집 후 [저장]

  또는 메모장으로 propagate_branches.yaml 파일을 직접 열어서 수정
