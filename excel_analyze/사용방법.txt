=====================================================
  Excel 데이터 구조 분석 툴 - 사용 방법
=====================================================

[ 최초 설치 (한 번만) ]

1. Python 설치
   - https://www.python.org/downloads/ 에서 Python 3.11 이상 다운로드
   - 설치 시 "Add Python to PATH" 체크박스 반드시 선택!

2. install.bat 더블클릭
   - 자동으로 필요한 프로그램을 설치합니다
   - "설치 완료!" 메시지가 나오면 성공

-----------------------------------------------------

[ 매번 실행할 때 ]

   run.bat 더블클릭
   → 브라우저에서 프로그램이 자동으로 열립니다

   종료: 열린 검은 창(터미널)을 닫으세요

-----------------------------------------------------

[ 사용 방법 ]

1. 왼쪽 사이드바 → [🔄 파싱 실행] 버튼 클릭
2. 카테고리별 테이블 탭에서 데이터 확인
3. FK 관계 목록 탭에서 테이블 간 연결 확인
4. 제외하고 싶은 파일은 "제외 파일 관리"에서 추가
